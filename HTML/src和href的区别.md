# src 和 href 的区别

## 是什么
`src` 和 `href` 都是 HTML 的特定元素的专有属性。

### 官方定义：
**href：超链接所指向的 `URL`。链接不限于基于 HTTP 的 `URL`——它们可以使用浏览器支持的任何 `URL` 协议**

简单来说，href 就是点击链接后会跳转的网页，href 连接的不一定要是网址 URL ，只要是浏览器能解析的地址就可以。

href 是 `hyper Reference` （超链接引用）的缩写，表示将要跳转或链接的地址。

**src：指定要使用的外部资源的 `URL`。**

src 是 `source` （来源）的缩写。表示要加载并显示/执行的外部资源。

## 区别
1. 引用方式不同

src 会将资源加载嵌入当前页面中，像 img 的 src 链接的资源会当作页面的一个元素嵌入到当前页面。

href 会将当前页面与其他网页/资源产生链接，以跳转形式连接而是不是将资源显式嵌入。

2. 阻塞行为不同

浏览器解析到 src 时会停止解析直到 src 的资源下载、执行完毕；

href 是异步加载的不会阻塞浏览器解析，它是 `<link>` 、`<a>`等元素的专有属性，所以这也是为什么推荐使用 `link` 而不是 `@import`。
