# 实现居中

## 一、 水平居中
### 1. 行内元素
```css
.content {
  text-align: center;
}
```

### 2. 块元素
```css
.content {
  /* 水平垂直居中： margin: auto; */
  height: 100px;
  width: 100px;
  margin-left: auto;
  margin-right: auto;
}
```

若有多个块元素要水平居中：
```css
.father {
  display: flex;
  margin-left: auto;
  margin-right: auto;
}
```
**注意**: 父盒子应设置为 `flex` 弹性布局，让子盒子都在主轴（默认是横轴）上排列，否则无法实现水平居中。

### 3. flex弹性布局
```css
.content {
  display: flex;
  justify-content: center;
}
```
此处的 `justify-content` 表示主轴上（默认为横轴）的对齐方式，可以通过 `flex-direction` 修改主轴，而 `align-content` 表示交叉轴的对齐方式。

### 4. grid布局
```css 
.content {
  display: grid;
  justify-content: center;
}
```

### 5. 绝对定位
```css
.content {
  position: absolute;
  left: 50%;
  transition: translateX(-50%)
}
```
如果盒子宽度已知也可以用负边距实现：
```css
.content {
  position: absolute;
  width: 100px;
  left: 50%;
  margin-left: -50px;
}
```
**拓展**：说到负边距则想到负边距的两种情况（标准文档流中）：

（1）`margin-left` 和 `margin-top` 为负数，则盒子自身会向左/上移动

（2）`margin-right` 和 `margin-bottom` 为负数，则与该盒子相邻的盒子会向左/上移动

若是绝对定位 `position: absolute;` 则上下左右都影响自身。

## 二、 垂直居中
### 1. 行内元素
`line-height`等于自身盒子高度：
```css
.content {
  height: 30px;
  line-height: 30px;
}
```

### 2. 块元素
```css
.content {
  height: 100px;
  width: 100px;
  margin-top: auto;
  margin-bottom: auto;
}
```

### 3. flex弹性布局
```css
.content {
  display: flex;
  align-content: center;
}
```
此处的 `justify-content` 表示主轴上（默认为横轴）的对齐方式，可以通过 `flex-direction` 修改主轴，而 `align-content` 表示交叉轴的对齐方式。

### 4. grid布局
```css 
.content {
  display: grid;
  align-content: center;
}
```

### 5. 绝对定位
```css
.content {
  position: absolute;
  top: 50%;
  transition: translateY(-50%)
}
```
如果盒子宽度已知也可以用负边距实现：
```css
.content {
  position: absolute;
  height: 100px;
  top: 50%;
  margin-top: -50px;
}
``` 

## 三、 水平垂直居中
就是把上述二者结合起来就可以了，不做赘述。
