## 选择器
**选择器优先级：** `!important` > 内联样式 > 类选择器、伪类选择器、属性选择器 > 标签选择器、伪元素选择器

### 通用选择器
用 `*` 表示选择全部元素：
```css
* {
  background-color: #ccc;
}
```

### 元素选择器（标签选择器）
```css
p {
  font-size: 25px;
}
```

### 类选择器
```css
.content {
  width: 100px;
}
```

### ID选择器
```css
#my-id {
  width: 100px;
}
```

### 组合选择器
**直接**连接两个或多个类名，用以选择同时具有这些类的元素。

**注意：**不需要加逗号也不想需要空格，加逗号表示选择多个元素。
```css
.my-class.special {
  width: 100px;
}
```

### 多重选择器
用逗号将多个类名分开

### 后代选择器
用空格将两个类名分开表示选择元素中的后代元素。
```css
.parent .child {
  width: 100px;
}
```

### 相邻选择器
用`+`连接多个类名表示选择相邻的元素:
```css
.item-1 + div {
  width: 100px;
}
```
表示选择与`.item-1`相邻的所有`div`。

### 子选择器
使用`>`作为组合符合，选择父元素的直接子元素。
```css
.item-1 > div {
  width: 100px;
}
```
表示选择.item-1的直接子元素的div。

### 通用兄弟选择器
使用`~`作为连接符表示选择第一个选择器**之后**的所有与它**同级**的兄弟元素。与相邻选择器不同的是不要求直接相邻。
```HTML
<div>
  <p>第一个段落</p>
  <h2>标题</h2>
  <p>第二个段落</p>
  <p>第三个段落</p>
</div>
```
在这个模块中 `.h2 ~ p` 选中的是第二个和第三个段落。

### 猫头鹰选择器
用于选择所有有前置兄弟元素的元素：
```css
* + * {
  width: 100px;
}
```

### 属性选择器
用于选择含有某种属性的元素。
