# TCP与UDP

TCP和UDP都是**传输层**协议。

1. **TCP：**全称Transmission Control Protocol，即传输控制协议。是一种**面向连接的**、**可靠的**、**基于字节流**的传输层通信协议。

**特点**：

- **面向连接**

发送数据之前必须在两端建立连接（TCP三次握手），为数据的可靠传输打下了基础。





2. **UDP：**全称User Datagram Protocol，即用户数据报协议，是一种**无连接**的传输层协议。UDP有不提供数据包分组、组装和不能对数据包进行排序的缺点，也就是说当报文发送后无法得知是否安全完整送达。



### TCP和UDP区别

|     特性     |           TCP            |       UDP        |
| :----------: | :----------------------: | :--------------: |
|     连接     |         面向连接         |      无连接      |
|     可靠     | 可靠（顺序、重传、校验） |      不可靠      |
|   流量控制   | 有（拥塞控制、滑动窗口） |        无        |
| 数据传输单位 |          字节流          |  报文（数据报）  |
|   适用场景   |        文件传输等        | 视频会议、直播等 |
