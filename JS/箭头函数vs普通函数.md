# 箭头函数与普通函数的区别

### （1）语法更简洁
直接用箭头连接参数和大括号括起来的函数体。

如果只有一个参数可以不写小括号，若没有参数就只写一个小括号。

### （2）没有自己的this
箭头函数继承自己作用域上一层的 `this` ，没有自己的 this 。

所以也可以知道箭头函数的 this 不会被 `apply()`, `call()`, `bind()` 改变。

### （3）不能用作构造函数
`new` 一个函数的过程：
- 创建一个对象
- 将对象的`.__proto__`指向构造函数的 `prototype` 属性（也就是将构造函数的 this 指向对象）
- 为对象添加属性和方法
- 返回对象

如果 new 了一个箭头函数，会在第二步就报错，因为箭头函数没有 `prototype` 属性，也没有自己的 this 。

### （4）没有 arguments 对象
arguments 是函数自动创建的类数组对象，里面包含了调用函数时传入的所有实参。
